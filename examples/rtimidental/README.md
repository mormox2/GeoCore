# RTimi Dental GeoCore Example

This example demonstrates how RTimi Dental can use GeoCore to manage structured dental knowledge.

It includes:
- Patient-facing dental Knowledge Objects
- Dental Entities
- Graph relationships
- Professional review citations
- Media metadata
- Validation pipeline
- Static export generation

## Commands

```bash
npm run validate
npm run export
npm run inspect
```

## Safety

This example does not include private patient data.

Dental content is educational and does not replace a personalized clinical examination.

## Expected outputs

The export command generates:
* Markdown
* JSON
* JSON-LD
* search-index.json
* llms.txt
* llms-full.txt
* sitemap.xml
* manifest.json
