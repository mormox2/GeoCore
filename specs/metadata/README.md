JSON Schema, field registry, and examples for the Metadata Engine.

## Status

Schemas to be added. All metadata MUST conform to GC-SPEC-0004 layers (§4–§10) and canonical requirements (§14).

## Six layers

1. Identity
2. Editorial
3. Semantic
4. Technical
5. SEO
6. AI/GEO
