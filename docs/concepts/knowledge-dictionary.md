# Knowledge Dictionary

A **Knowledge Dictionary** is a controlled vocabulary: terms, definitions, aliases, and cross-references.

It aligns with [`knowledge/glossary/`](../../knowledge/glossary/) and [`knowledge/taxonomy/`](../../knowledge/taxonomy/).

## Difference from glossary files

- **Dictionary** (concept) — the model and rules
- **`knowledge/glossary/`** — live term instances

Semantic metadata references glossary terms via `glossaryTerms` — see [GC-SPEC-0004 §7](../../specs/0004-metadata-engine.md).
