# Knowledge Collection

A **Knowledge Collection** is an ordered group of [Knowledge Objects](knowledge-object.md) with purpose and navigation logic.

Defined in [GC-SPEC-0003 §13](../../specs/0003-knowledge-graph.md).

## Principle

A Collection is not merely a category. It has **order**, **purpose**, and **navigation rules**.

## Use cases

- Guides and learning paths
- Documentation sections and help centers
- Topic clusters and courses
- API bundles and LLM context windows

## Examples

```text
collection_implantology_guide
collection_dawajin_delivery_management
collection_customer_balance_documentation
```
