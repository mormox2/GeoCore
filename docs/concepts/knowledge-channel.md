# Knowledge Channel

A **Knowledge Channel** is a distribution surface where rendered knowledge reaches an audience.

Examples: website section, API namespace, LLM context bundle, mobile app module, voice skill.

## Relationship to renderers

A channel selects which [Knowledge Renderer](knowledge-renderer.md) runs and which [Knowledge Collection](knowledge-collection.md) it exposes.
