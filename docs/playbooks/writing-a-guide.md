# Playbook — Writing a Guide output

A guide is rendered output — a sequential reading experience over a [Knowledge Collection](../concepts/knowledge-collection.md).

## Steps

1. Define learning objective and audience.
2. Select or create knowledge objects for each section.
3. Order objects in a collection (prerequisites first).
4. Apply the guide renderer (TOC, progress, cross-links).
5. Publish to the target [Knowledge Channel](../concepts/knowledge-channel.md).

## Do not

- Treat "guide" as a separate storage format in `docs/` or `knowledge/`
